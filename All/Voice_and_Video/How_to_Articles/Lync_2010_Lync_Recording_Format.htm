Lync 2010 recording format

The recording format in Microsoft Lync 2010 communications software provides a rich viewing experience for the presenter who created the recording. From within Microsoft Lync 2010 Recording Manager, you can view the audio, video, instant messaging (IM) conversations, application sharing, Microsoft PowerPoint presentations, whiteboard, and polling portions of the Lync 2010 recording just as they were experienced during the Lync 2010 meeting or conversation. You can also hide any of these features so that you can focus on other parts of the Lync 2010 recording. With indexing, you can listen to a particular speaker or view specific content, and you can also access a list of attendees and download attachments.

Lync 2010 saves recordings in the default location C:\Users\username\Lync Recordings. You can change this location from within Lync 2010.

To specify where you want to save recordings:

  1. In the Lync main window, click the Options button ( ), and then click File Saving.
  2. Under Lync recordings, click Browse.
  3. Navigate to the folder where you want recordings to be saved, and then click OK.

If you want to make a recording available to others, you can save all or parts of the Lync 2010 recording in Windows Media file format. This option consolidates all of the recorded content that you want to share into a single Windows Media video, which you can publish to others. You can create Windows Media version of the recording from within Lync Recording Manager. For details, see Recording Manager.
