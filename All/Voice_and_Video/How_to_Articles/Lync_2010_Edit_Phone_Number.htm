Edit a phone number

Use the Edit Phone Number dialog box to add your own mobile, home, or other phone number to Microsoft Lync 2010 communications software.

When specifying phone numbers, use the format for international phone numbers. Type the country code followed by the local number. Phone numbers should contain only the digits 0123456789. You do not need to type parentheses or hyphens.

The following table shows examples of how to convert a local phone number into the format Lync requires.
