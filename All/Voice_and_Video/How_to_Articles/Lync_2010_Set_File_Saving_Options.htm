Set File Saving options

Use File Saving options to specify where you want to save files that you receive in conversations and recordings that you make of meetings.

What do you want to do?
  • Open File Saving options
  • Save transferred files
  • Save recorded meetings

Open File Saving options

To view File Saving options, do the following:

  1. Click Start, click All Programs, click Microsoft Lync, and then click Microsoft Lync 2010.
  2. In the Lync main window, click the Options button, and then click File Saving.

Top of Page

Save transferred files

You can save files that others send to you in Lync 2010 to review, work on, or share later.

  • Open File Saving options, and then, under File transfer, click Browse to select the folder that you want to keep received files in.

Top of Page

Save recorded meetings

You can record an online meeting and save it to play back later or send to someone.

  • Open File Saving options, and then, under Lync recordings, click Browse to select the folder that you want to keep recorded files in.
