Select the method for configuring your connection settings

Use the Advanced Connections Settings dialog box to select either automatic or manual configuration of connections to the communications service, if and as directed by your system administrator.

What do you want to do?
  • Use automatic configuration
  • Use manual configuration

Use automatic configuration

Open Microsoft Lync 2010 communications software, click the Options button, click Personal, click the Advanced button, and then click Automatic configuration to let Microsoft Lync Server 2010 communications software automatically configure your connection settings.

Use manual configuration

Open Lync 2010, click the Options button, click Personal, click the Advanced button, click Manual configuration, and then use the information provided by your system administrator to configure the following information:

  • Internal server name or IP address Type the fully qualified domain name (FQDN) or the IP address of your Lync Server 2010.
  • External Server name or IP address Type the FQDN (or the IP address of your Lync Server 2010, Edge Server.
  • Connect using Click the appropriate connection protocol for your network.
