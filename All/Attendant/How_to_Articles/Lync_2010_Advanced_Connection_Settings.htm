Advanced Connection Settings

Use the Advanced Connections Settings dialog box to select either automatic or manual configuration of connections to the communications service.

What do you want to do?
  • Automatic configuration
  • Manual configuration

Automatic configuration

  • Click to let Microsoft Lync 2010 Attendant communications software configure your connection settings automatically.

Top of Page

Manual configuration

  • Click to configure your connection settings manually. To obtain the necessary information, see your system administrator.
    • Internal server name or IP address      Type the FQDN (fully qualified domain name) or the IP address of your Microsoft Lync Server 2010.
    • External server name or IP address      Type the FQDN (fully qualified domain name) or the IP address of your Lync Server 2010, Edge Server.
    • Connect using      Click the appropriate connection protocol for your network.
