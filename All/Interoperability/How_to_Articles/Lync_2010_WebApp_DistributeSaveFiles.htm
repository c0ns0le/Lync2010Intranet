Distribute and save files for later viewing

Meeting participants can distribute files, including documents and images, by uploading them to the meeting’s Attachments space. The other participants can then save the files to their computer for later use.

 Important    Administrative settings and your role (as presenter or attendee) in a particular meeting determine whether you can upload files to the meeting or access files that are added by others.

What do you want to do?
  • Save a file from the meeting
  • Add a file to the meeting
  • Allow only meeting presenters to save your file
  • Rename a file
  • Delete a file from the meeting

Save a file from the meeting

You receive a notification when a meeting participant adds a file to the meeting.

  • To save the file, click the Add or view attachments button , click the file name, select a location, change the file name (if you want), and then click Save.

Add a file to the meeting

To distribute a file to meeting attendants, do the following:

  1. In the meeting window, click the Add or view attachments button , click Add Attachment, and then navigate to the file you want to distribute.
  2. Double-click the file.
    The file uploads to the Attachments space.
  3. When it finishes uploading, click Close.
    The other participants are notified that a file attachment is available.

Allow only meeting presenters to save your file

You may want to distribute the file to presenters only.

  • In the meeting window, click the Add or view attachments button , click the arrow next to the file name, click Make Available To, and then click Presenters.

Top of Page

Rename a file

You may want to rename a file to distinguish it from other meeting files.

  1. In the meeting window, click the Add or view attachments button , click the arrow next to the file name, and then click Rename.
  2. Type the new name, and then click OK.

Top of Page

Delete a file from the meeting

When you delete a file from the meeting it is no longer available to be saved by meeting participants. However this will not affect any copies of the file that were already saved.

  • In the meeting window, click the Add or view attachments button , click the arrow next to the file name, and then click Remove from Meeting.
